# Molotov Paperboy

A browser clone of the arcade classic *Paperboy*, re-themed: instead of a paperboy
tossing newspapers into mailboxes, you're a kid bombing down "Cyber Boulevard" on a
glowing **Sirron** board, lobbing **Molotov cocktails** at **Cyber Trucks**.

## Play

Open `index.html` in any modern browser (no build, no dependencies).

- **← / →** — steer across the street
- **↑ / ↓** — speed up / slow down
- **SPACE** — throw a Molotov
- Touch devices get on-screen buttons.

## Rules

- Each Cyber Truck takes **2 hits** to torch (+100). A non-lethal hit is +15.
- **Don't ram** trucks or curbside junk (cones, hydrants, bins) — each crash costs a life.
- Off-road junk past the curb is harmless; only what's on the road can hit you.
- Score also ticks up with distance. "HEAT" rises with score, speeding traffic up.
- 3 lives. Game over when they run out.

Single self-contained file — `index.html` holds all markup, styles, and game logic.
