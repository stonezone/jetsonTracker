03 Network

Two phone routes, one backend

The app can work while the Orin is USB tethered to the phone or when the Orin is a Wi-Fi/hotspot client with a static address.

USB tether route

Default API: http://172.20.10.8:8088/api/v1. This is used when the phone is plugged into the Orin and the tether interface is reachable.

Wi-Fi/hotspot route

Default API: http://192.168.1.155:8088/api/v1. This is used when the Orin is on Wi-Fi/hotspot and the phone can reach that static address.

Routing rule

Polling tries configured candidates and updates activeRoute from the responding URL. When Wi-Fi/custom is active, the app periodically probes tether again so plugging the phone back in can restore the USB path without a rebuild.

06 Cinematic Zoom

Auto-frame, default off

Cinematic Zoom is a bounded zoom policy, not a second tracker. It uses locked YOLO person boxes and leaves pan/tilt authority alone.

Enable path

  1. Connect on USB or Wi-Fi and verify Live feed.
  2. Open PTZ, tap Start Auto, and verify owner moves to autonomous state.
  3. Open Tools > Tune. Confirm the Cinematic Zoom card is visible.
  4. Enable ptz.cinematic_zoom_enabled and start with ptz.zoom_target_frac=0.50.
  5. Watch the Live overlay: zoom should move only from a locked person box, then return to hold inside the deadband.

Failure rules

  • No person box: zoom holds.
  • No lock: zoom holds.
  • Manual zoom nudge: auto-zoom suppresses briefly, pan/tilt tracking continues.
  • Bad framing or twitching: disable the toggle and retest pan/tilt first.

Enable by shell

$ curl -X POST http://127.0.0.1:8088/api/v1/config/hot \
-H "Content-Type: application/json" \
-d '{"patch":{"ptz.cinematic_zoom_enabled":true,"ptz.zoom_target_frac":0.50}}'

Disable by shell

$ curl -X POST http://127.0.0.1:8088/api/v1/config/hot \
-H "Content-Type: application/json" \
-d '{"patch":{"ptz.cinematic_zoom_enabled":false}}'
07 Safety

Controls are separated

Stop PTZ is not Emergency Stop. Restart is not Resume. The operator has to read state before repeating commands.

Emergency

Kill

Global latch. Stops motion and blocks future motion until Resume.

Hold

Stop PTZ

Manual hold. Keeps the backend reachable and the live feed available.

Auto

Start Auto

Requests autonomous owner. Refuses while killed.

Restart

Service

Stops PTZ and restarts WaveCam service; used after restart-only config changes.
