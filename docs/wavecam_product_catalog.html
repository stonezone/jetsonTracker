Built for one job: keep the rider in frame.

WaveCam trims the system down to the parts that matter in the field: camera movement, target detection, status visibility, and reliable capture.

Vision-primary tracking

YOLO person detection and an orange/red HSV cue work together. GPS helps point and reacquire; vision owns framing when the target is visible.

Real PTZ camera control

The camera is controlled directly from the Orin over Ethernet using validated raw VISCA. No STM32, no stepper drivers, no DIY gimbal stack.

LoRa GPS ready

The Watch/iPhone GPS path is being retired. The next GPS lane is Wio Tracker L1 Lite nodes with Meshtastic-style position packets into the Orin.

Phone-first dashboard

Start follow, stop follow, preview the sub-stream, nudge PTZ, check services, watch disk space, and see what the tracker thinks it is doing.

Jetson inference core

Jetson Orin Nano runs the camera loop, OpenCV color tracker, YOLO/TensorRT path, and operator dashboard. Recording stays copy/remux friendly.

Safety by ownership

Only one autonomous PTZ owner should run at once. Manual moves are blocked while follow owns the camera, and KILL/STOP remain first-class controls.

Field workflow without the laptop dance.

Designed around the real setup sequence: power it, see if it is healthy, aim it, start it, ride.

Set the rig

Tripod on the beach, Ethernet to the PTZ camera, Orin powered, camera LAN static, internet over Wi-Fi or phone tether.

Confirm health

The dashboard checks camera route, preview, service state, recording space, and whether any tracker already owns PTZ.

Calibrate view

Aim at the waterline, center the rider target, tune orange cue if needed, then lock the starting pose for stop/restore.

Start follow

Vision follow tracks the subject, zoom only follows YOLO person boxes, and color-only frames hold zoom instead of hunting.
